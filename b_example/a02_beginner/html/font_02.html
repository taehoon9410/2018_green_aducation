굵기에대해 되짚어 보자

서체는 디자이너 또는 과녈ㄴ회사에서 제작,
이때, 해당 서체의 기울기 또는 굵기는 제작을 했을경우에만 적용이 가능!!

굵기의 표기방법 또한, 디자인/회사의 제작여부에 따라 이름이 달리질 수 있다. 기본 용어로 사용한다면:ultra thin,thin,light,regular,normal,bold,bolder.... 수치로 사용한다면 100~200(thin)~400(normal,regular)~600(bold)~800(bolder)~900

웹폰트를 사용해 보자!

CDN방식으로 적용(웹상에 존재하는 컴퓨터/별도 디렉토리에 다운받지 않고,인터넷으로 연결하여 사용하는기술)
구글 웹폰트를 이용하여 적용해보기 http://fonts.google.com

@font-face { font-family:"만들 서체를 직접 가져와서 링크" src: url() font-weight:굵기 설정; font-style:기울기 설정; }
